# Calculator  

## A browser-based calculator built with HTML, CSS, and JavaScript.
Part of The Odin Project Foundations curriculum.

[The Odin Project](https://www.theodinproject.com/lessons/foundations-calculator)  
[Live Demo](https://ccs0708.github.io/calculator/)

## Features
- Addition, subtraction, multiplication, division
- Chained calculations
- Decimal support
- Clear and delete buttons
- Keyboard support

### Keyboard support available
- `0` - `9`
- `.`
- `=` **:** `Enter` or `=` key
- `Clear` **:** `c` key
- `Backspace`
- `+` **:** `Shift` + `=` key
- `-` **:** `-` key
- `*` **:** `x` or `Shift` + `=` key
- `/` **:** `/` key
